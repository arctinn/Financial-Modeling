# Financial Modeling Project📈

An open-source, educational project/engine designed to bridge the gap between pure financial mathematics and programmatic execution for my own learning purpose. 

This repository is built for entry level developers and analysts like me who want to understand the foundational accounting concepts, the mathematical equations of valuation, and how to execute them efficiently using Python.

## Project Architecture

This project is broken down into total of 3 progressive phases:

### [Phase 1: Foundations](./docs/phase_1_foundations/1_system_mechanics.md)
Understanding how a business measures success before writing any code.
* The Income Statement, Balance Sheet, and Cash Flow Statement.
* Bridging accounting profit (Net Income) to valuation cash (Unlevered Free Cash Flow).

### [Phase 2: The Models](./docs/phase_2_models/)
Deep dives into the mathematical equations and Python implementations of the four core valuation methodologies:
1. Discounted Cash Flow (DCF)
2. Leveraged Buyout (LBO)
3. Accretion / Dilution (M&A)
4. Relative Valuation (Comps)

### [Phase 3: Integration](./docs/phase_3_integration/)
*Coming Soon:* Connecting the standalone Python models into an automated data pipeline.
